## ğŸš€ Installation & Setup

Since this repository includes the virtual environment and database, setup is extremely fast.

### 1. Clone the Repository
```bash
git clone [https://github.com/NOT-in-SYNC/recipe-planner.git](https://github.com/NOT-in-SYNC/recipe-planner.git)
cd recipe-planner

```

### 2. Activate the Environment

* **Windows:**
```bash
venv\Scripts\activate

```


* **Mac/Linux:**
```bash
source venv/bin/activate

```



### 3. Run the Server

No need to migrate! The database is already set up.

```bash
python manage.py runserver

```

### 4. Open in Browser

Visit **https://www.google.com/search?q=http://127.0.0.1:8000/** to start planning your meals!
